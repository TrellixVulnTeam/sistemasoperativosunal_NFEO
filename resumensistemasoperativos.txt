Sistemas Operativos ¿Que es? Caracteristicas, tipos, funcion, TODO sobre S.O.:

"Un sistema operativo es un conjunto de programas que controla la ejecucion del software
de aplicación y actúa como una interfaz entre el usuario y el hardware."Tambien es el 
que permite la instalacion de software de aplicación.
Facilita el manejo de los dispositivos, no se requieren muchos conocimientos para él manejo.
Caracteristicas:
-Habilidad para evolucionar. Las actualizaciones no deben interferirir en el funcionamiento
del SO.
-Comunicar los dispositivos.
-Organiza datos para el acceso rapido y seguro, da prioridades.
-Maneja las comunicaciones de red.
-Facilita el acceso a dispositvos de entrada y de salida, recuperacion de errores.
Tipos:
-Con interfaz grafica tactil.
-Multitareas.
-Monotarea.
-Monousario
-Multiusuario
Funciones y entorno:
-Gestion del hardware y facilitar el trabajo del usuario. 
-Administración de recursos.
-Utilierias que complementan las tareas.
